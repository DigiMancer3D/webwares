The Hash Key Cutter (HKC) will allow you to take any hash (specifically designed for base64 hashes) & hashlinks (specially designed for IB 1.0 hashed-links) and cut them into restorable master keys. HKC can create non-restorable keys as well so use the chart below to determine which keys you may want to use or need to save for only the Master key is able to restore the full given hash/hashlink.

	--------------------------------------------------------------------------------------------------------------------------------------
	View Key (Verifiable): Not Alone
	--------------------------------------------------------------------------------------------------------------------------------------
	Priv Key (Proovable Ownership): Not Alone
	--------------------------------------------------------------------------------------------------------------------------------------
	Mater Key (Restoring): Yes

The webapp will be split into 2 sections. For most it will be oriented left or right, but for some and mobile users it may be top or bottom and we are in-fact talking about the input fields. Everthing inputted and displayed will be within these two input fields. Next is how to use each section.

 Left/Top (Make a Key Section):
--------------------------------
 1) Paste a hash or hashlink in the correct input field
 2) Click "Cut Keys"
 3) Copy all text from right field (after it appears), these are your keys
 4) Paste Keys into a text document
 5) Save your Keys to a harddrive or mobile device like a thumbstick


======================================================================================================================================
======================================================================================================================================


 Right/Bottom (Restore a Key Section):
----------------------------------------
 1) Paste a Master Key in the correct input field
 2) Make sure the identifier "Mv#" is not present in the input field start
 3) If you see Mv# at the beggining of your master key, remove it from the input field; If you don't see Mv# at the beggining of your master key, continue
 4) Click "Restore Master"
 5) Copy all text from the left field (after it appears), this is your restored hash
 6) Paste your hash into a text document
 7) Save your Hash to a harddrive or mobile device like a Micro-SD card
