----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
SuPosXt is Superpositioning text to a type of End-2-End Encrypted Font based on mapping matematical complexities to a specialized grid matrix to allow for faux-quantum like mechanics in binary that is fast enough to encode per click of the user input.
----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------


The entire idea is based on the Talk Model seen in the hit sitcom, "Full House" and utlizes some mathematical ideas from Futurama's Theorem.
